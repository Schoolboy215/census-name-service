Census names (1.0.0)

Download OpenAPI specification:

API that generates realistic American names from public census bureau and social security administration data

Get a random first name.

Get a random first name with optional filters for sex, year of birth, and state as well as rarity.

Authorizations:
ApiKeyAuth
Request Body schema:

Optional filters

sex
string
Enum: "M" "F"
yob
number [ 1910 .. 2023 ]
state
string
Enum: "AL" "AK" "AZ" "AR" "CA" "CO" "CT" "DE" "DC" "FL" "GA" "HI" "ID" "IL" "IN" "IA" "KS" "KY" "LA" "ME" "MD" "MA" "MI" "MN" "MS" "MO" "MT" "NE" "NV" "NH" "NJ" "NM" "NY" "NC" "ND" "OH" "OK" "OR" "PA" "RI" "SC" "SD" "TN" "TX" "UT" "VT" "VA" "WA" "WV" "WI" "WY"
percentile
integer [ 1 .. 100 ]
top
string
Enum: "true" "false"
